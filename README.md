# IQ Test - 知能指数テスト

5つの認知領域を測定する標準化IQテストアプリです。

## 機能

- **図形推理** — マトリクスパターン認識（6問）
- **数列推理** — 数列の規則性発見（6問）
- **言語类比** — 言葉の関係性推論（6問）
- **論理推理** — 論理的帰結の導出（6問）
- **空間認知** — 図形の回転・変換理解（5問）

## スコアリング

- 平均IQ 100、標準偏差15の正規分布に基づく換算
- 難易度加重スコアリング
- 領域別パフォーマンス分析

## 起動方法

```bash
npm install
npm run dev
```

ブラウザで http://localhost:5173 を開いてください。

## 注意

本テストは娯楽・自己評価目的です。正式な心理検査（ウェクスラー式等）の代替ではありません。
